USING TRACK CHANGES IN MS WORD

Find below the steps to use the function Track Changes in MS Word.

The Track Changes function allows you to add temporary changes to a text to facilitate editing and proofreading.

  1. Go to the Review tab.
  2. Click on Track Changes.
  3. Edit the text as you like.
  4. Right-click on any change to accept or reject.

You will see the changes underlined and colored so that you can easily identify them and accept or reject them.
